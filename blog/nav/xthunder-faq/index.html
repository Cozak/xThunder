xThunder是什么？
xThunder是调用迅雷/QQ旋风/快车等下载器的火狐3.5+的轻量级扩展，体积只有FlashGot的1/6。

xThunder不是什么？
xThunder本身不是下载软件，只是充当火狐与下载器之间的桥梁，额外具有链接解码的功能。

xThunder会不会更改名字或图标？
由于xThunder已经提交到AMO，对应了一个Name/ID/Statistics。因此不会改名了，图标也暂时不会修改。

xThunder会不会移植到Chrome浏览器？
近期内没有时间移植至Chrome，可以使用Chrome类似扩展Downloaders。

我应该安装哪个版本？
一般而言，您总应该安装xThunder最新版，有更多的特性和更少的BUG，并始终保持轻量级。

为什么Mozilla主页显示的不是最新版？
Mozilla主页显示的是审核通过的版本，所以最新版可能还在审核中，你可以在版本历史中找到。

出现错误对话框提示“由于应用程序配置不正确，应用程序未能启动”？
早前的操作系统缺少VC2005运行库，请安装微软的VC补丁。

出现错误对话框提示“Cannot open job file”？
无法找到临时文件。如果操作系统用户名是中文的，请尝试修改为英文。

无法调用绿色版或修改版迅雷“Call Thunder error, please check if it was properly installed”？
注册表信息不完整导致无法调用，可尝试下面几种方法。
    1）下载我写的 批处理程序 解压后将BAT放在迅雷的根目录下运行。
    2）尝试迅雷菜单项“工具->浏览器支持->迅雷作为IE默认下载工具”修复此问题。
    3）用命令行cmd注册组件
      regsvr32 “迅雷目录\Comdlls\ThunderAgent.dll” (迅雷5)
      regsvr32 “迅雷目录\BHO\ThunderAgent.dll” (迅雷7)
      △检查注册表项值 “Path” = “迅雷目录\Thunder.exe”，不存在则新建Path
      x86 路径：[HKEY_LOCAL_MACHINE\SOFTWARE\Thunder Network\ThunderOem\thunder_backwnd
      x64 路径：[HKEY_LOCAL_MACHINE\SOFTWARE\Wow6432Node\Thunder Network\ThunderOem\thunder_backwnd]
    4）win7用户请重新安装在系统盘默认位置，避免UAC问题。
    5）不想捣鼓注册表的，可以通过xThunder选项中“新增”下载器指定迅雷可执行程序路径

无法调用迅雷官方精简版 “Call ThunderLite error, please check if it was properly installed”？
注册表信息不完整导致无法调用，可尝试下面几种方法。
    1）下载我写的 批处理程序2 解压后将BAT放在迅雷精简版的根目录下运行。
    2）用命令行cmd注册组件
      regsvr32 “迅雷精简版目录\BHO\ThunderAgent.dll”
      △检查注册表项值 “Path” = “迅雷精简版目录\MiniThunder.exe”，不存在则新建Path
      x86 路径：[HKEY_LOCAL_MACHINE\SOFTWARE \Thunder Network\MiniXLOEM\thunder]
      x64 路径：[HKEY_LOCAL_MACHINE\SOFTWARE\Wow6432Node\Thunder Network\MiniXLOEM\thunder]
    3）win7用户请重新安装迅雷精简版在系统盘默认位置。
    4）不想捣鼓注册表的，可以通过xThunder选项中“新增”下载器指定迅雷精简版可执行程序路径

无法调用QQ旋风“Call QQDownload error, please check if it was properly installed”？
在旋风之后安装或更新迅雷破坏了QQ旋风的BHO，regsvr32 QQIEHelper01.dll或重装QQ旋风。

选择下载器为QQ旋风却调用了迅雷？
与本扩展无关。迅雷的“破解专用链接”插件会修改注册表导致此问题，不要安装此插件。

第一次调用BitComet无法成功新建任务？
启动慢导致的问题，BitComet选项->界面外观->默认显示的详细信息选项卡->更改为“起始页”外的其他值。

专用链接无法破解？
国内下载站纷繁复杂，专用链不一定按照标准格式，只支持Alexa 10000以内的大型下载站。

某些链接破解后下载却没有速度？
少部分资源是存放在迅雷/旋风/快车服务器上的，因此只有使用它们的软件才能下载对应的资源。

为何左键点击迅雷/快车/磁力等专用链接直接弹出迅雷进行下载？
在xThunder设置中“左键直接下载”可以取消你不想关联的专用链接。

为何左键点击.exe/.zip等普通文件或论坛附件直接弹出迅雷进行下载？
在xThunder设置中取消“关联普通文件类型”或者删除你不想关联的后缀（如.mp3;）。

使用中出现异常情况，重装xThunder仍然不行？
因为Firefox扩展配置保留在profile，即使重装扩展也不会删除，在xThunder选项中点“恢复默认配置”即可。

不定时弹出迅雷下载.js等文件？
与本扩展无关。在迅雷设置中取消SPI多浏览器支持，如还有问题则取消监视浏览器。

诺顿杀毒软件报“智能网络威胁”，自动删除xThunder.exe？
本扩展绝对绿色，完全开源且通过Mozilla审核和Softpedia认证。属诺顿误报，添加信任即可。

使用wine-thunder，下载前会弹出一个对话框？
vi ~/.wine-thunder/drive_c/Program Files/Thunder/thunder.vbs 删除第7行 msgBox url 即可。

迅雷5没有选择下载目录对话框就直接下载？
在迅雷菜单中勾选“配置->高级->通过IE右键菜单’使用迅雷下载’添加任务”。
